Viewing NIF Files

Applies to: Epic Mickey 2

Most NIF files can be viewed in Noesis, a program designed to open game graphics files.

Notes:

  • Not sure if some NIF files contain 3D models.
  • Some, like mickey.nif, return some sort of Python error when opened.
  • This sadly does not seem for work for nif_wii files from the first game, renamed or not.
